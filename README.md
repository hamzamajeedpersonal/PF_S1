# 🚀 C++ Programming Fundamentals Suite
A comprehensive collection of C++ programs covering core programming concepts, ranging from basic arithmetic and logic to complex nested loops and mathematical series.
## 📌 Project Overview
This repository serves as a practical archive of fundamental programming logic. Each file is designed to solve a specific problem, demonstrating mastery of:
 * **Decision Making:** if-else, switch-case
 * **Iterative Logic:** for, while, do-while loops
 * **Mathematical Algorithms:** Series, Prime numbers, Palindromes
 * **Data Handling:** Variable manipulation and Swapping logic

## 🛠️ How to Run
To compile and run any of these programs, ensure you have a C++ compiler (like g++) installed.
 1. **Clone the repository:**
   ```bash
   git clone https://github.com/YourUsername/RepositoryName.git
   
   ```
 2. **Navigate to the folder:**
   ```bash
   cd RepositoryName
   
   ```
 3. **Compile a specific file:**
   ```bash
   g++ Basic_Calc.cpp -o Basic_Calc
   
   ```
 4. **Execute the program:**
   ```bash
   ./Basic_Calc
   
   ```
